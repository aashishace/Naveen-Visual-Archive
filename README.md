# Naveen Visual Archive

A cinematic portfolio website for a Director of Photography & Editor, designed to showcase visual storytelling through immersive video backgrounds and interactive elements.

### Tech Stack
- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Video:** Vimeo Player SDK
- **AI:** Google Gemini Integration

### Local Development
```bash
npm install
npm run dev
```
