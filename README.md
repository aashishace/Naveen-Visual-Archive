# Naveen's Cinematography Portfolio

A high-performance, immersive portfolio website for a Director of Photography & Editor. Built with React, Vite, Tailwind CSS, and Framer Motion.

## Features
- **Immersive Video Backgrounds**: Seamless Vimeo integration with custom controls.
- **Dark Mode Aesthetic**: High-contrast, cinematic visual style.
- **Interactive Elements**: Custom cursor, magnetic buttons, and haptic feedback.
- **AI Chat Interface**: Integrated Gemini-powered assistant ("AI COMM").
- **Performance Optimized**: Fast loading with optimized assets and code splitting.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env` (if applicable) or configure `services/geminiService.ts`.
3. Run the app:
   `npm run dev`
